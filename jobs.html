Software Developer

#SWD93
$55,000 – $85,000 Full-Time Local Reports to: Senior Software Engineer

Design, develop, and maintain software applications, ensuring high performance, scalability, and usability.

Key Responsibilities

  • Develop and maintain web or desktop applications using industry-standard technologies.
  • Write clean, maintainable, and well-documented code.
  • Debug, troubleshoot, and fix application issues.
  • Collaborate with UI/UX designers and product managers on feature requirements.
  • Implement APIs, integrate databases, and participate in code reviews.

Required Skills Preferred Skills

  3. Understanding of SQL databases (MySQL/PostgreSQL).
